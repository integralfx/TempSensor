# TempSensor
STM32 L452RE temperature and humidity sensor using the RHT03 which is displayed on a 16x2 LCD

# 5V Tolerant Pins
| Digital Pin | Port & Pin | 5V Tolerant? |
| ----------- | ---------- | ------------ |
| D0  | PA3  | N |
| D1  | PA2  | Y |
| D2  | PA10 | Y |
| D3  | PB3  | Y |
| D4  | PB5  | Y |
| D5  | PB4  | Y |
| D6  | PB10 | Y |
| D7  | PA8  | Y |
| D8  | PA9  | Y |
| D9  | PC7  | Y |
| D10 | PB6  | Y |
| D11 | PA7  | Y |
| D12 | PA6  | Y |

| D15 | PB8  | Y |
